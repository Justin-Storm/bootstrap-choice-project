Whats is web design?

The Honors Web Design WDPP course is the first course in the CTE Web Design Pathway Program. In this course, students will learn the fundamentals of the internet, coding principles, Hypertext Markup Language (HTML) and Cascading Styles Sheets (CSS). This course will prepare students to create and maintain professional static websites using current industry tools and emerging trends and technologies. Students will design and create functional web pages according to specifications using HTML, CSS, Bootstrap, and emerging trends and technologies. In addition, students will solve problems collaboratively and creatively in a professional environment, while developing transferable, marketable skills in preparation for careers in the field of web design.
